## 3. 研究方法对比

### 3.1 方法使用频率

| 方法 | CER | 经济研究 | 差异 |
|------|-----|----------|------|
| **DID** | 20.6% | 22.2% | ≈ 相当 |
| **Panel Data** | **25.4%** | 7.4% | CER +18% |
| **Experiment** | **17.7%** | 0% | CER 特有 |
| **IV** | 12.2% | 10.0% | ≈ 相当 |
| **Fixed Effects** | 5.2% | **17.4%** | JJYJ +12% |
| **GE Model** | 2.0% | **13.9%** | JJYJ +12% |
| **Game Theory** | 1.7% | **7.8%** | JJYJ +6% |
| **Text/NLP** | 0.7% | **6.5%** | JJYJ +6% |

### 3.2 解读

**CER 方法特点**：
- 大量使用 **Panel Data**（面板数据分析）
- **实验方法** 是显著特色（行为经济学、实验室实验）
- 偏好 **简化形式估计**（reduced-form）

**经济研究 方法特点**：
- 偏好 **一般均衡模型**（理论建模）
- 大量使用 **博弈论** 框架
- 更多 **文本分析/NLP** 方法
- **固定效应模型** 使用更频繁

### 3.3 方法论风格

| 维度 | CER | 经济研究 |
|------|-----|----------|
| **理论 vs 实证** | 实证为主 | 理论建模多 |
| **识别策略** | 准实验设计 | 结构估计 |
| **数据处理** | 面板回归 | 固定效应 |
| **创新方法** | 实验经济学 | 大语言模型文本分析 |

## 4. 行业焦点对比

### 4.1 行业分布差异

| 行业 | CER | 经济研究 | 差异 |
|------|-----|----------|------|
| **Manufacturing** | 6.3% | **20.7%** | JJYJ +14.4% |
| **Finance** | 1.4% | **11.6%** | JJYJ +10.2% |
| **Services** | 0.2% | **5.5%** | JJYJ +5.3% |
| **Digital Economy** | 0.4% | **4.3%** | JJYJ +3.9% |
| **Agriculture** | **3.9%** | 5.1% | ≈ 相当 |
| **Banking** | **3.6%** | 2.2% | CER +1.4% |
| **Healthcare** | **3.6%** | 1.7% | CER +1.9% |
| **Education** | **3.2%** | 1.4% | CER +1.8% |
| **Energy** | 0.7% | **2.7%** | JJYJ +2.0% |
| **Real Estate** | 0.6% | **2.2%** | JJYJ +1.6% |

### 4.2 解读

**经济研究 行业焦点**：
- **制造业 (Manufacturing)**: 占比最高（20.7%），产业升级、智能制造是核心议题
- **金融 (Finance)**: 第二大行业（11.6%），金融改革、风险防控
- **服务业 (Services)**: 现代服务业发展
- **数字经济 (Digital Economy)**: 平台经济、数字化转型热点
- **能源 (Energy)**: 碳中和、新能源转型

**CER 行业焦点**：
- **Healthcare**: 健康经济学、医疗体制改革
- **Education**: 教育回报、人力资本投资
- **Banking**: 银行信贷、普惠金融
- **Agriculture**: 农村发展、农业经济

### 4.3 行业-领域对应关系

| 行业 | CER 对应 JEL | 经济研究对应 JEL |
|------|-------------|------------------|
| Manufacturing | F, L | **L, O, D** |
| Finance | G | **G, E** |
| Healthcare | **I** | I |
| Education | **I, J** | I |
| Agriculture | Q | Q |
| Digital Economy | O | **O, L, D** |

### 4.4 时间趋势

**经济研究行业趋势**：
- **Manufacturing**: 稳定高位（15-29%），2025-06 达峰值
- **Finance**: 2025-12 后显著上升（达 17%）
- **Agriculture**: 2026-02 突增至 11%

**CER 行业趋势**：
- **Healthcare**: 持续上升，2026-02 达 6%
- **Banking**: 波动大（0-5%）
- **Education**: 相对稳定（2-5%）

### 4.5 行业焦点差异的背后

**经济研究聚焦实体经济**：
- 反映中国产业政策研究需求
- 制造业转型升级是国家战略重点
- 数字经济研究响应新发展理念

**CER 聚焦社会领域**：
- 面向国际学术界，关注社会议题
- 健康、教育是跨国可比的研究主题
- 劳动市场研究有丰富微观数据支撑

## 6. 核心结论

### 6.1 期刊定位差异

| 维度 | CER | 经济研究 |
|------|-----|----------|
| **目标读者** | 国际学术界 | 国内政策/学术界 |
| **研究风格** | 实证微观 | 理论+政策 |
| **方法偏好** | 准实验+实验 | 结构模型+博弈 |
| **选题倾向** | 国际/劳动/健康 | 产业/金融/体制 |
| **行业焦点** | 医疗/教育/银行 | **制造业/金融/数字经济** |

### 6.2 互补性

两刊形成良好的互补：
- **CER**: 适合发表国际比较、劳动市场、实验研究、社会领域研究
- **经济研究**: 适合发表产业政策、中国特色理论、金融研究、实体经济研究

### 6.3 共同趋势

1. **DID 方法** 是两刊共同的主流方法 (~20%)
2. **数字经济** 相关研究（AI、数字化）在两刊都有增长
3. **绿色/环境** 主题两刊都有关注
4. **制造业** 在两刊都是重要行业（但经济研究占比更高）
